# Kubernetes 
In order to give a good demonstration of Kubernetes, it is useful to have materials that you can use for a demonstration. 
This folder contains these materials for demonstration purposes. 
Each folder contains a YAML file with a deployment of an image on top of Kubernetes. 
Where possible, a BASH shell script or even a Windows command line script has also been added. 

## Flask 
Deployment of flask-demo image on Kubernetes. 
If you would like to know more about the flask demo image, please refer to docker within the demo repository.

## NGINX
This is a demonstration of the initial rollout of nginx version 14 as two replicas on top of Kubernetes. 
That is step 1. The next step (step 2) is to upgrade nginx from version 14 to version 16. 
The final step (step 3) is to increase the number of replicas from two to four.
