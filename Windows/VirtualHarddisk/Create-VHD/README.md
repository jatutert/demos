# Create virtuel machine from any Windows ISO file 

Hyper-ConvertImage version 10.2 is used for conversion of any ISO file to VHD(X) file 
More info on this Powershell script can be found at https://www.powershellgallery.com/packages/Hyper-ConvertImage/10.2

After conversion VMWare tools is copied onto the new VHD(x) so that it can be used within the new virtual machine.
